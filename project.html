Movie Rank Project

The Movie Title Project is a simple webpage built using only HTML, designed to display a list of movie titles in a clean and organized format. The project focuses on practicing basic HTML structure and understanding how to present information using semantic elements.

The page includes headings, paragraphs, and lists to showcase different movies, along with short descriptions or release years. Since the project uses only HTML, it highlights how content can be structured without styling or interactivity

Key Features
  • Created using HTML only
  • Displays multiple movie titles in a structured layout
  • Uses headings (h1, h2, h3) to separate sections
  • Beginner-friendly and easy to understand
  • Demonstrates basic web page structure and semantic HTML usage
Purpose of the Project
  • Understand correct HTML hierarchy
  • Organize information clearly
  • Build simple static pages
  • Practice writing clean and readable HTML code

Birth Day Invite Project

The Birthday Invite Project is a simple and beginner-friendly webpage created using only HTML. The goal of the project is to design a digital birthday invitation that clearly displays event details such as the celebrant's name, date, time, venue, and activities.

This project helps practice how to structure a webpage using headings, paragraphs, lists, and links. It also demonstrates how to use semantic elements to organize information in a clean and readable format.

Key Features
  • Uses only HTML (no CSS or JavaScript required)
  • Displays event information in a clear and organized layout
  • Uses headings (h1, h2, h3) to separate sections
  • Includes a list of things guests should expect and bring
  • Includes an RSVP link using an anchor tag
  • Beginner-friendly and good for learning HTML basics

HTML Portfolio Project

The HTML Portfolio Project is a basic personal website created using only HTML. Its purpose is to present information about yourself in a structured and professional way without using any CSS or JavaScript. The project includes essential portfolio sections such as an introduction, list of skills, completed projects, contact details, and hobbies.

This project focuses on building a clean and organized layout using semantic HTML elements. Headings, paragraphs, lists, and links are used to display content clearly. Even without styling, the project shows how HTML alone can create a simple, readable, and functional portfolio.

Key Features
  • Built using HTML only
  • Contains key portfolio sections:
    • About Me
    • Skills
    • Projects
    • Hobbies
    • Contact
  • Shows understanding of headings, lists, paragraphs, and links
  • Beginner-friendly and easy to expand later with CSS or JavaScript
Purpose of the Project
  • Practice writing clean and structured HTML code
  • Understand layout organization using semantic tags
  • Build a simple foundation for a future full portfolio website
  • Present personal information clearly without relying on styling
